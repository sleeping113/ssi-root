ssi-root
========

*用Eclipse创建的maven聚合性的基于springmvc + spring + ibatis的ssi框架

   1，ssi-root 根项目，其他项目都继承于它
   
   2，ssi-model 用户存放实体类
   
   3，ssi-dao 用户存放持久层代码
   
   4，ssi-service 用于存放业务层代码
   
   5，ssi-controller 用于存放控制层代码
   
   
   
架构大体如下：

	--ssi-root
		--ssi-model
		--ssi-dao
		--ssi-service
		--ssi-controller
